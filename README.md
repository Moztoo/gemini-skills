# Gemini CLI Skills

This repository is a collection of custom skills designed to extend the capabilities of the Gemini CLI. Each skill in this repository provides specialized instructions and tools to help Gemini perform specific tasks more efficiently.

## What are Skills?

Skills are "packages" of instructions, scripts, and reference documents that teach Gemini new workflows. They allow you to customize and enhance the agent for your specific needs, automating complex tasks and integrating with your favorite tools.

## Available Skills

*   **[git-assistant](./git-assistant/)**: An assistant to manage Git repositories. It helps with common operations like committing, pushing, and even creating a new GitHub repository and connecting it when one isn't configured.

## Usage

To use these skills, clone this repository into your local Gemini skills directory. Gemini will automatically discover and load them, making them available for you to use in your conversations.
